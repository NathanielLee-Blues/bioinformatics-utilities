# Input data

This folder contains real bioinformatics input files used to demonstrate the utility scripts.

## Files

| File | Description |
|---|---|
| `reference/ecoli_rel606_reference.fasta` | E. coli REL606 reference genome copied from the variant-calling workflow project |
| `reads/SRR2584866_R1_1000_reads.fastq` | First 1,000 reads from a real paired-end E. coli FASTQ file |
| `reads/SRR2584866_R2_1000_reads.fastq` | First 1,000 reads from the matching read 2 FASTQ file |
| `variants/SRR2584866_filtered_variants.vcf` | Filtered VCF generated by the variant-calling workflow project |

The FASTQ files are subsetted so the repository remains lightweight while still using real sequencing reads.
